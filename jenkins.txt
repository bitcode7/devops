# Jenkins Using Docker Complete Lab Guide (Ubuntu)

## Step 1 — Update Ubuntu Packages

```bash
sudo apt update
```

# Refreshes Ubuntu package list

---

# Step 2 — Install Docker

```bash
sudo apt install docker.io -y
```

# Installs Docker Engine

---

# Step 3 — Verify Docker Installation

```bash
docker --version
```

# Displays Docker version

---

# Step 4 — Start Docker Service

```bash
sudo systemctl start docker
```

# Starts Docker service

---

# Step 5 — Enable Docker on Boot

```bash
sudo systemctl enable docker
```

# Automatically starts Docker after reboot

---

# Step 6 — Check Docker Status

```bash
sudo systemctl status docker
```

# Displays Docker service status

---

# Step 7 — Download Jenkins Docker Image

```bash
sudo docker pull jenkins/jenkins:lts
```

# Downloads Jenkins LTS image from Docker Hub

---

# Step 8 — View Downloaded Images

```bash
sudo docker images
```

# Lists Docker images

---

# Step 9 — Run Jenkins Container

```bash
sudo docker run -d \
-p 8080:8080 \
-p 50000:50000 \
--name jenkins \
jenkins/jenkins:lts
```

# Creates Jenkins container
# 8080 → Jenkins web interface
# 50000 → Jenkins agents communication

---

# Step 10 — Check Running Containers

```bash
sudo docker ps
```

# Shows running containers

---

# Step 11 — Open Jenkins in Browser

Visit:

```text
http://localhost:8080
```

OR

```text
http://<ubuntu-ip>:8080
```

# Opens Jenkins dashboard

---

# Step 12 — Get Jenkins Admin Password

```bash
sudo docker exec jenkins cat /var/jenkins_home/secrets/initialAdminPassword
```

# Displays Jenkins setup password

---

# Step 13 — Install Suggested Plugins

Inside Jenkins:
- Click:
  ```text
  Install suggested plugins
  ```

# Installs common Jenkins plugins

---

# Step 14 — Create Jenkins Admin User

Create:
- username
- password

# Creates Jenkins admin account

---

# Step 15 — Install Build Pipeline Plugin

Go to:

```text
Manage Jenkins → Plugins → Available Plugins
```

Search and install:
- Build Pipeline Plugin
- jQuery Plugin

# Adds pipeline visualization support

---

# Step 16 — Install Git on Ubuntu

```bash
sudo apt install git -y
```

# Installs Git

---

# Step 17 — Verify Git Installation

```bash
git --version
```

# Displays Git version

---

# Step 18 — Install Python

```bash
sudo apt install python3 -y
```

# Installs Python

---

# Step 19 — Verify Python Installation

```bash
python3 --version
```

# Displays Python version

---

# Step 20 — Create GitHub Repository

Example file:

```python
print("Hello from Jenkins Pipeline")
```

# Push repository to GitHub

---

# Step 21 — Create Jenkins Job 1

Inside Jenkins:
- Click:
  ```text
  New Item
  ```

Name:

```text
git_job1
```

Select:

```text
Freestyle Project
```

# Creates first Jenkins job

---

# Step 22 — Configure GitHub Repository

Go to:

```text
Source Code Management → Git
```

Repository URL:

```text
https://github.com/YOUR_USERNAME/YOUR_REPO.git
```

# Connects Jenkins with GitHub repository

---

# Step 23 — Add Build Step

Go to:

```text
Build Steps → Execute Shell
```

Add:

```bash
python3 demo.py
```

# Executes Python program during build

---

# Step 24 — Save Job

Click:

```text
Save
```

# Stores Jenkins job configuration

---

# Step 25 — Create Job 2

Create another:
- Freestyle Project

Name:

```text
job2
```

# Creates second Jenkins job

---

# Step 26 — Configure Downstream Trigger

Go to:

```text
Build Triggers
```

Enable:

```text
Build after other projects are built
```

Projects:

```text
git_job1
```

Select:

```text
Trigger only if build is stable
```

# Automatically triggers job2 after successful job1

---

# Step 27 — Add Shell Script to Job 2

Go to:

```text
Build Steps → Execute Shell
```

Add:

```bash
python3 -c "import datetime; print(datetime.datetime.now())"
```

# Prints current timestamp

---

# Step 28 — Save Job 2

Click:

```text
Save
```

# Stores second job configuration

---

# Step 29 — Create Pipeline View

Dashboard:
- Click:
  ```text
  +
  ```

Choose:

```text
Build Pipeline View
```

Name:

```text
pipeline_test
```

Initial Job:

```text
git_job1
```

# Creates pipeline visualization

---

# Step 30 — Run Pipeline

Inside `git_job1`:
- Click:
  ```text
  Build Now
  ```

# Starts Jenkins pipeline

---

# Expected Pipeline Flow

```text
git_job1 → job2
```

# job2 automatically runs after job1

---

# Step 31 — View Console Output

Open:

```text
Build History → Console Output
```

# Displays:
# - Git clone logs
# - Python execution
# - timestamps
# - build logs

---

# Step 32 — Stop Jenkins Container

```bash
sudo docker stop jenkins
```

# Stops Jenkins container

---

# Step 33 — Start Jenkins Container Again

```bash
sudo docker start jenkins
```

# Restarts Jenkins container

---

# Step 34 — View Jenkins Logs

```bash
sudo docker logs jenkins
```

# Displays Jenkins container logs

---

# Step 35 — Remove Jenkins Container

```bash
sudo docker rm -f jenkins
```

# Deletes Jenkins container permanently

---

# Important Jenkins + Docker Concepts

| Concept | Meaning |
|----------|----------|
| Jenkins | Automation server |
| Docker Image | Blueprint/template |
| Container | Running Jenkins instance |
| Pipeline | Connected Jenkins jobs |
| Build | Execution of Jenkins job |
| Plugin | Additional Jenkins feature |
| Console Output | Build logs |
